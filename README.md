# WJXConfusion
    
## 介绍
    代码混淆,小概率避免被逆向!

## 版本
    1.0.3

## 准备工作
### 修改脚本
*  新建文件夹
*  放入climbMethod.sh和placeCode.h
*  拖入要添加代码的项目
*  修改项目参数folder_path是要添加代码的文件夹路径
   folder_path=${project_path}/${project_name}/${project_name}/xxxx
* 把placeCode.h拖入工程引入预编译文件 要放在靠前位置
   

## 使用方法
* 使用终端进入WJXConfusion文件
* 执行:sh climbMethod.sh
